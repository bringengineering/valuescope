# Glossary — 용어 사전

| 용어 | 영문 | 정의 |
|---|---|---|
| 잠재총수입 | GPR (Gross Potential Rent) | 모든 호실 100% 임대 시 연간 총 임대수입 |
| 유효총수입 | EGI (Effective Gross Income) | GPR − 공실손실 − 미수금 + 기타수입 |
| 운영비 | OPEX | 재산관리·청소·수선·보험·공용전기 등 운영 지출 (원리금·감가·세금 제외) |
| 순영업소득 | NOI | EGI − OPEX. 부채·감가·소득세 차감 전 |
| 안정화 NOI | Stabilized NOI | 안정화(장기) 공실률을 적용한 NOI |
| 원리금균등 | Equal Payment | 매월 동일한 원리금을 상환하는 방식 |
| 원금균등 | Equal Principal | 매월 동일한 원금 + 잔액 이자 |
| 만기일시 | Interest-Only / Bullet | 이자만 납부, 원금은 만기 일시상환 |
| 거치기간 | Grace Period | 원금 상환 없이 이자만 내는 기간 |
| 담보인정비율 | LTV (Loan-to-Value) | 대출금 ÷ 담보가치 |
| 총사업비대출비율 | LTC (Loan-to-Cost) | 대출금 ÷ 총사업비 |
| 부채상환비율 | DSCR (Debt Service Coverage Ratio) | NOI ÷ 연간 원리금상환액 |
| 부채수익률 | Debt Yield | NOI ÷ 대출금 |
| 자본환원율 | Cap Rate | NOI ÷ 자산가격 |
| 현금수익률 | Cash-on-Cash | 세전현금흐름 ÷ 초기 자기자본 |
| 자기자본배수 | Equity Multiple | 총 회수현금 ÷ 초기 자기자본 |
| 내부수익률 | IRR | 보유기간 현금흐름의 순현가를 0으로 만드는 할인율 |
| 손익분기 점유율 | Break-even Occupancy | 비용·원리금을 감당하는 최소 임대율 |
| 실질 레버리지 | Real Leverage | (금융부채 + 임차보증금 반환의무) ÷ 자산가치 |
| 절대 상한가 | Walk-away Price | 목표 수익률·최소 DSCR·최소 CoC를 모두 만족하는 최대 매입가 |
| 하드 스톱 | Hard Stop | 총점으로 상쇄할 수 없는 투자 금지 조건 |
